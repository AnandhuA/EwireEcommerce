# Ewire Ecommerce

Flutter E-commerce Application

## Features

- Product Listing
- Product Details
- Search Products
- Add to Cart
- Cart Persistence using Hive
- Swipe to Delete Cart Items
- Dark / Light Theme
- Pull to Refresh
- Shimmer Loading
- Product Reviews
- Responsive UI

## Tech Stack

- Flutter
- Provider
- Dio
- Hive
- Cached Network Image

## Setup

1. Clone repository
2. Run flutter pub get
3. Run flutter run
